Django CBV Inspector

  • @charleswdenton
  • @mellenthenerd
  • @mjtamlyn
  • @rjmunro

Django's Class Based Views are great, however it's a well knwon fact that their documentation is quite lacking and following the code can get confusing when you have classes with 8 parents.

The Django CBV Inspector team set out to fix this with a two pronged attack of a management command to inspect the code (using Python's inspect module) and a front end that allowed easy navigation of a class' methods be they inherited or not.
